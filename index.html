這個看板怎麼用？

1
看進度
打開網頁就是最新版，免登入。橫條＝任務的起訖日期；顏色看下方圖例 （綠＝完成、藍＝進行中、灰＝未開始、橘＝卡住等待）。虛線＝今天。 表格可上下左右捲動，左欄與日期列會固定不動。
2
想反映意見（例：日期不對、有做過但沒記到）
點那一列任務 → 跳出視窗 → 按「💬 回報這個任務」→ 寫下你想說的 → 送出。 板主看到後會更新。（回報需要免費 GitHub 帳號，第一次會請你註冊）
3
不會有「改壞」的問題
這個頁面對你是唯讀的——你在板上做什麼都不會影響別人看到的內容。 正式變更只有板主能發布。
